GtaVHelper

GtaVHelper is a lightweight, high-performance utility designed for GTA V modders and texture artists. It streamlines the process of converting standard images into game-ready textures and ensures RPF archives are correctly signed and fixed for game compatibility.

Built with Python 3.8.10 and PySide2, it provides a clean GUI for batch processing and archive management.
🚀 Features

    Batch DDS Conversion: Select multiple PNG, JPG, or JPEG files and convert them all at once.

    Advanced Compression: Supports industry-standard formats:

        DXT1: Best for simple textures without transparency.

        DXT5: Ideal for complex textures with alpha channels.

        BC7: High-quality modern compression (recommended for GTA V).

    Mipmap Generation: Toggleable checkbox to automatically generate mipmap chains.

    ArchiveFix Integration: Quickly sign and fix GTA V .rpf archives to prevent game crashes after modding.
